# 묵주기도 (Rosary PWA)

가톨릭 묵주기도를 도와주는 설치형 웹앱(PWA)입니다. 설치 없이 브라우저에서 바로 쓸 수 있고,
휴대폰 홈 화면에 추가하면 앱처럼 전체화면으로 동작합니다.

## 주요 기능
- **요일별 신비 안내** — 오늘 요일에 맞는 신비(환희·빛·고통·영광)를 자동으로 보여줍니다.
- **기도문 전문** — 성호경, 사도신경, 주님의 기도, 성모송, 영광송, 구원송(파티마 기도), 마침 기도까지 전체 텍스트 제공.
- **진행 단계 추적** — 5단 진행바와 10개 묵주알 표시로 지금 어디까지 바쳤는지 한눈에 확인.
- **한 단씩 자동 진행** — '자동 진행'을 켜면 설정한 시간(2~30초)마다 다음 기도로 넘어갑니다. 화면 탭/다음 버튼으로 수동 진행도 가능.
- **이어서 기도하기** — 중간에 나가도 같은 날이면 마지막 위치에서 이어서 바칠 수 있습니다(localStorage 저장).
- **오프라인 지원** — 서비스워커 캐시로 인터넷 없이도 동작.

## 요일별 기본 신비
| 요일 | 신비 |
|------|------|
| 월·토 | 환희의 신비 |
| 화·금 | 고통의 신비 |
| 수·일 | 영광의 신비 |
| 목 | 빛의 신비 |

홈 화면의 '신비 선택'에서 다른 신비로 바꿀 수도 있습니다.

## 사용 / 배포
- 로컬 확인: 이 폴더에서 `python3 -m http.server` 실행 후 브라우저로 접속.
  (서비스워커/매니페스트는 `file://`가 아닌 http(s)에서 동작합니다.)
- 배포: 폴더 전체를 정적 호스팅(GitHub Pages 등)에 올리면 됩니다.

## 파일 구성
- `index.html` — 앱 본체(단일 파일, HTML/CSS/JS 포함)
- `manifest.json` — PWA 매니페스트
- `sw.js` — 서비스워커(오프라인 캐시)
- `icon-192.png`, `icon-512.png` — 앱 아이콘 (원하면 교체 가능)

## 조작
- 기도문 영역을 탭하거나 **다음** 버튼 → 다음 단계
- **‹** → 이전 단계
- 키보드: → / Space(다음), ←(이전), Esc(처음으로)
- 🔔 → 진동 켜기/끄기 (지원 기기 한정)

> 기도문은 한국 천주교 표준 기도서를 따랐습니다.
